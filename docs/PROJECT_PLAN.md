# Vaani-AI Project Plan

## Goal
Build an AI-powered mobile application that bridges communication between deaf/hard-of-hearing and hearing individuals.

## Core Features

1. Sign → Speech
2. Speech → Text
3. Conversation Mode
4. Quick Phrase Board
5. Multi-language Support
6. Offline AI Inference

## Tech Stack

Frontend:
- Flutter

Backend:
- Python Flask

AI:
- MediaPipe
- TensorFlow Lite
- LSTM

Database:
- Firebase Firestore

Authentication:
- Firebase Auth
